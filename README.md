# LIMPANOME - Plataforma de Gestão de Clientes

Plataforma web elegante e profissional para gestão de clientes com autenticação PIN 2FA, painel de cliente, painel administrativo e design brutalista moderno.

## Características

- ✅ Autenticação segura com PIN de 6 dígitos via email
- ✅ Painel do cliente para gerenciar clientes e afilhados
- ✅ Painel administrativo com dashboard de estatísticas
- ✅ Hierarquia de afilhados
- ✅ Design brutalista elegante
- ✅ Banco de dados com 5 tabelas
- ✅ Notificações automáticas por email
- ✅ Proteção de rotas por autenticação

## Stack Técnico

- React 19 + TypeScript + Tailwind CSS 4
- Express 4 + tRPC 11
- MySQL/TiDB com Drizzle ORM
- Manus OAuth + JWT
